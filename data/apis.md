# Top Books on APIs

- **[Designing Web APIs](https://www.oreilly.com/library/view/designing-web-apis/9781492026914/)** - *Brenda Jin et al.* - *2018* - Building APIs that developers love; REST, GraphQL, and design patterns.
- **[REST in Practice](https://www.oreilly.com/library/view/rest-in-practice/9780596805838/)** - *Jim Webber et al.* - *2010* - Hypermedia and systems architecture with REST.
- **[API Design Patterns](https://www.manning.com/books/api-design-patterns)** - *JJ Geewax* - *2021* - Standard patterns for REST, gRPC, and GraphQL APIs.
- **[Automating API Delivery](https://www.oreilly.com/library/view/automating-api-delivery/9781633438781/)** - *Ikenna Nwaiwu* - *2024* - APIOps: design reviews, deployment automation, OpenAPI, and linting.
- **[Building Microservices, 2nd Ed.](https://www.oreilly.com/library/view/building-microservices-2nd/9781492034018/)** - *Sam Newman* - *2021* - API design, integration, and deployment in microservice architectures.
