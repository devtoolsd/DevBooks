# Top Books on Spring

- **[Spring in Action, 6th Ed.](https://www.manning.com/books/spring-in-action-sixth-edition)** - *Craig Walls* - *2022* - The definitive practical guide to the Spring ecosystem.
- **[Spring Boot in Action](https://www.manning.com/books/spring-boot-in-action)** - *Craig Walls* - *2016* - Spring Boot for production applications; auto-configuration, actuators, and deployment.
- **[Pro Spring 6](https://www.apress.com/gp/book/9781484280791)** - *Iuliana Cosmina et al.* - *2022* - In-depth coverage of Spring Framework 6 and Spring Boot 3.
- **[Learning Spring Boot 3.0](https://www.oreilly.com/library/view/learning-spring-boot/9781803233307/)** - *Greg L. Turnquist* - *2022* - Build reactive and production-ready Spring Boot applications.
