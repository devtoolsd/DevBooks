# Top Books on PowerShell

- **[Learn PowerShell Scripting in a Month of Lunches, 2nd Ed.](https://www.manning.com/books/learn-powershell-scripting-in-a-month-of-lunches-second-edition)** - *Don Jones, Jeffery Hicks, James Petty* - *2020* - 27 bite-sized lessons on scripting, pipeline, error handling, and professional practices.
- **[PowerShell in Depth, 2nd Ed.](https://www.manning.com/books/powershell-in-depth-second-edition)** - *Don Jones, Jeffery Hicks, Richard Siddaway* - *2014* - Practical desk reference for Windows admins; 40+ chapters on production scripting.
- **[Mastering PowerShell Scripting, 4th Ed.](https://www.packtpub.com/product/mastering-powershell-scripting-fourth-edition/9781800206540)** - *Chris Dent* - *2020* - Advanced topics: classes, modules, async, GUIs, error handling, and automation.
- **[Learn Windows PowerShell in a Month of Lunches, 4th Ed.](https://www.manning.com/books/learn-windows-powershell-in-a-month-of-lunches-fourth-edition)** - *James Petty, Don Jones, Jeffery Hicks* - *2022* - Core PowerShell for daily administration; one lesson per day.
