# Top Books on SQL

- **[Learning SQL, 4th Ed.](https://www.oreilly.com/library/view/learning-sql-3rd/9781492057604/)** - *Alan Beaulieu* - *2023* - Window functions, materialized views, JSON; graded exercises and solutions.
- **[Practical SQL, 2nd Ed.](https://www.nostarch.com/practical-sql-2nd-edition)** - *Anthony DeBarros* - *2022* - Storytelling with PostgreSQL; spatial queries and hands-on labs.
- **[SQL Cookbook, 2nd Ed.](https://www.oreilly.com/library/view/sql-cookbook-2nd/9781492077435/)** - *Anthony Molinaro & Robert de Graaf* - *2020* - Real-world SQL recipes; window functions and CTEs.
- **[Effective SQL](https://www.oreilly.com/library/view/effective-sql/9780134578897/)** - *John Viescas, Douglas Steele, Ben Clothier* - *2017* - 61 patterns for better, optimized queries.
- **[SQL in a Nutshell, 4th Ed.](https://www.oreilly.com/library/view/sql-in-a/9781492088851/)** - *Kevin Kline, Regina Obe, Leo Hsu* - *2022* - Reference for ANSI SQL and major database products.
