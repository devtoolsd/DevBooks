# Top Books on Cloud

- **[Designing Data-Intensive Applications](https://dataintensive.net/)** - *Martin Kleppmann* - *2017* - Storage, streams, and consistency across distributed and cloud systems.
- **[Cloud Architecture Patterns](https://www.oreilly.com/library/view/cloud-architecture-patterns/9781449343220/)** - *Bill Wilder* - *2012* - Patterns for building scalable, reliable applications on cloud platforms.
- **[Which Cloud?](https://www.wiley.com/en-us/Which+Cloud%3A+A+Developer%27s+and+Architect%27s+Guide+to+Comparing+AWS%2C+Microsoft+Azure%2C+and+Google+Cloud-p-9781119760115)** - *Vishal Agrawal* - *2021* - Comparing AWS, Azure, and GCP at the service and feature level.
- **[Infrastructure as Code, 3rd Ed.](https://www.amazon.com/Infrastructure-Code-Dynamic-Systems-Cloud/dp/1098114671)** - *Kief Morris* - *2021* - Managing cloud and server infrastructure with code.
