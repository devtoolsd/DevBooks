# Top Books on Node.js

- **[Node.js Design Patterns, 3rd Ed.](https://www.nodejsdesignpatterns.com/)** - *Mario Casciaro & Luciano Mammino* - *2020* - Patterns, best practices, and idioms for server-side Node.js.
- **[Get Programming with Node.js](https://www.manning.com/books/get-programming-with-node-js)** - *Jonathan Wexler* - *2019* - Build web servers and APIs with Node.js and Express.
- **[Node.js Web Development, 5th Ed.](https://www.packtpub.com/product/node-js-web-development-fifth-edition/9781838987572)** - *David Herron* - *2020* - Full-stack Node.js: Express, REST, real-time, and deployment.
- **[Practical Node.js](https://www.apress.com/gp/book/9781484230381)** - *Azat Mardan* - *2018* - Building real-world Node.js applications and APIs.
