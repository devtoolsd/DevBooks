# Top Books on Docker

- **[The Ultimate Docker Container Book](https://www.oreilly.com/library/view/the-ultimate-docker/9781804613986/)** - *Dr. Gabriel N. Schenker* - *2023* - Docker fundamentals, Kubernetes, security, monitoring, and cloud deployment.
- **[Docker in Action, 2nd Ed.](https://www.manning.com/books/docker-in-action-second-edition)** - *Jeff Nickoloff & Stephen Kuenzli* - *2019* - Build, ship, and run containers; networking and orchestration.
- **[Using Docker](https://www.oreilly.com/library/view/using-docker/9781491915752/)** - *Adrian Mouat* - *2015* - Developing and deploying applications with Docker.
