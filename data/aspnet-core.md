# Top Books on ASP.NET Core

- **[ASP.NET Core in Action, 3rd Ed.](https://www.manning.com/books/asp-net-core-in-action-third-edition)** - *Andrew Lock* - *2023* - Building web applications and APIs with ASP.NET Core.
- **[Pro ASP.NET Core 8](https://www.apress.com/gp/book/9781484291766)** - *Adam Freeman* - *2023* - Comprehensive guide to ASP.NET Core 8 and the .NET 8 platform.
- **[C# 12 and .NET 8](https://www.amazon.com/12-NET-Modern-Cross-Platform-Development/dp/1837635870)** - *Mark J. Price* - *2023* - Modern C# and .NET including ASP.NET Core and web development.
