# Top Books on Google Cloud

- **[Google Cloud Platform in Action](https://www.manning.com/books/google-cloud-platform-in-action)** - *John J. Geewax* - *2018* - Build and launch scalable applications on GCP; storage, compute, serverless, Kubernetes.
- **[Google Cloud Cookbook](https://www.oreilly.com/library/view/google-cloud-cookbook/9781492092889/)** - *Rui Santos Costa & Drew Hodun* - *2021* - Practical recipes for GCP: microservices, AI, analytics, security, networking.
- **[Google Cloud Certified Associate Cloud Engineer Study Guide](https://www.amazon.com/Google-Cloud-Certified-Associate-Engineer/dp/1119812742)** - *Dan Sullivan* - *2021* - Prepare for the ACE certification; core GCP services and best practices.
