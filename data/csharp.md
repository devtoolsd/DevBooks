# Top Books on C#

- **[C# in Depth, 4th Ed.](https://csharpindepth.com/)** - *Jon Skeet* - *2019* - The book for understanding the why behind C# features, not just syntax.
- **[Concurrency in C# Cookbook, 2nd Ed.](https://www.amazon.com/Concurrency-Cookbook-Asynchronous-Multithreaded-Programming/dp/149205450X)** - *Stephen Cleary* - *2019* - Practical patterns for async, parallel, and reactive C# code.
- **[C# 12 and .NET 8](https://www.amazon.com/12-NET-Modern-Cross-Platform-Development/dp/1837635870)** - *Mark J. Price* - *2023* - Comprehensive, up-to-date coverage of modern C# and the .NET ecosystem.
- **[Programming C# 12](https://www.oreilly.com/library/view/programming-c-12/9781098158354/)** - *Ian Griffiths* - *2024* - C# 12 and .NET 8: generic math, pattern matching, modern polymorphism; for experienced programmers.
- **[Pro C# 10 with .NET 6](https://www.apress.com/gp/book/9781484278697)** - *Andrew Troelsen & Phil Japikse* - *2022* - Foundational principles and practices; OOP, ASP.NET Core, Entity Framework, Razor, Web APIs.
- **[Essential C#](https://essentialcsharp.com/)** - *Mark Michaelis with Eric Lippert* - *2023* - Complete tutorial and reference; best-practice guidelines; free online companion.
- **[CLR via C#, 4th Ed.](https://www.microsoftpressstore.com/store/clr-via-c-sharp-fourth-edition-9780735667457)** - *Jeffrey Richter* - *2012* - How the CLR and .NET runtime work; types, GC, threading, and assembly loading (classic deep dive).
- **[ASP.NET Core in Action, 3rd Ed.](https://www.manning.com/books/asp-net-core-in-action-third-edition)** - *Andrew Lock* - *2023* - Building web applications and APIs with ASP.NET Core.
