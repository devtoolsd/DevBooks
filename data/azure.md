# Top Books on Azure

- **[The Azure Cloud Native Architecture Mapbook, 2nd Ed.](https://www.oreilly.com/library/view/the-azure-cloud/9781805805052/)** - *Stéphane Eyskens* - *2025* - Architecture for infrastructure, data, AI, and security on Azure; AKS and solution design.
- **[Microsoft Azure Fundamentals Certification and Beyond](https://www.packtpub.com/product/microsoft-azure-fundamentals-certification-and-beyond/9781803236636)** - *Steve Miles* - *2022* - Azure services, identity, storage, compute, and certification prep.
- **[Azure for Developers](https://www.packtpub.com/product/azure-for-developers/9781788831948)** - *Kamil Mrzygłód* - *2018* - Building and deploying applications on Microsoft Azure.
