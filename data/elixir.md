# Top Books on Elixir

- **[Programming Elixir 1.6](https://pragprog.com/titles/elixir16/programming-elixir-1-6/)** - *Dave Thomas* - *2018* - The best starting point for Elixir's functional and concurrent programming model.
- **[Elixir in Action, 3rd Ed.](https://www.manning.com/books/elixir-in-action-third-edition)** - *Saša Jurić* - *2019* - Production-grade Elixir: OTP, concurrency, and building real fault-tolerant systems.
- **[Programming Phoenix 1.4](https://pragprog.com/titles/phoenix14/programming-phoenix-1-4/)** - *Chris McCord, Bruce Tate, José Valim* - *2018* - Building real-time web apps with Phoenix and LiveView.
